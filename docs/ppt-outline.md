# GameVault 演示 PPT 提纲

## 1. 项目背景

- 游戏玩家常常同时拥有愿望单、已购游戏、正在玩游戏和已通关游戏。
- GameVault 用数据库统一管理游戏收藏、状态、评分、消费和游玩进度。

## 2. 系统目标

- 支持用户登录和个人游戏数据管理。
- 支持游戏、平台、类型、标签和游玩日志的增删改查。
- 支持数据统计分析和可视化展示。
- 支持 Docker 一键启动，便于现场演示。

## 3. 技术架构

- 前端：React、TypeScript、Ant Design、ECharts。
- 后端：Python、FastAPI、SQLAlchemy。
- 数据库：MySQL 8。
- 部署：Docker Compose。
