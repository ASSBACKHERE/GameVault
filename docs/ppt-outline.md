## PPT 生成总提示词

请为一个数据库系统课程期末大作业生成一份中文演示 PPT，项目名称为：

**GameVault — 游戏愿望单与游玩进度分析系统**

这是一套基于 **Python FastAPI + MySQL + React + Docker Compose** 的全栈数据库 Web 应用系统。系统用于帮助游戏玩家管理个人游戏库、愿望单、游玩状态、平台、类型、标签、评分、消费金额和游玩日志，并提供统计分析图表。

请生成一份适合课堂展示和答辩的 PPT，要求如下：

1. 总页数：14 页。
2. 语言：中文。
3. 风格：深色高级游戏科技风，视觉关键词包括“星空、霓虹蓝紫、玻璃拟态、数据仪表盘、游戏封面卡片、光门、科技线条”。
4. 主色调：深蓝黑 `#0B1020`，霓虹紫 `#7C3AED`，科技蓝 `#2563EB`，青色 `#22D3EE`，强调绿 `#22C55E`。
5. 页面不要堆大段文字，每页突出 3-5 个核心点。
6. 每页需要包含标题、正文要点、视觉建议和演讲提示。
7. 需要突出数据库系统课程要求：需求分析、数据库设计、数据字典、实体完整性、参照完整性、增删改查、测试用例、系统演示。
8. 涉及图表时请使用清晰的数据可视化样式，例如卡片统计、饼图、折线图、ER 图、架构图、流程图。
9. 涉及截图时请使用占位说明，例如“此处放系统首页截图”“此处放 Adminer 数据库表截图”。
10. 整体呈现应像一个完成度较高的课程项目，而不是简单作业。
